# 看懂一下 (Kandong) — PRD v1

> 比赛 Demo · 目标：拿奖 · 受众：60+ 长辈 · 场景：识别诈骗截图/链接/文案

## 1. 一句话定位

老人看不懂的截图/链接，对着手机说一句话，AI 用大字+语音解释「这是什么、有没有风险、怎么办」，一张可分享给子女的「家人卡」。

## 2. 核心三条 (MUST)

1. **看一眼就懂**：上传截图 → 30s 内出「红/黄/绿」风险灯 + 一句话结论 + 3 条人话解释（≤20 字/条）+ 语音播报。
2. **断网也能演**：3 个内置 Demo 案例（灰/黑/白）走离线兜底 JSON，评委断网拔线照样跑。
3. **家人卡分享**：Canvas 生成 1080×1920 PNG，长按保存/微信发给子女，二维码回链。

## 3. 砍 (NOT in MVP)

- 用户账号体系
- 历史记录持久化（仅 sessionStorage）
- 多语言（仅简中）
- 复杂诈骗类型库（先 3 类：杀猪盘/钓鱼链接/虚假投资）
- 后台管理

## 4. P1 (Demo 后补)

- 真实 LLM 调用（豆包主 / GPT-4o 备）走云端，本次比赛默认走 mock + 离线 JSON
- 浏览器端 OCR（Tesseract.js）
- 家人卡微信分享 SDK

## 5. 页面 (4 屏)

| # | 路由 | 功能 | 关键元素 |
|---|------|------|----------|
| 1 | `/` | 首页 | 80px 标题 + 3 大按钮：拍照/相册/试一下 |
| 2 | `/scan` | 识别中 | 大圆 loading + 「正在看懂…」 + 语音同步 |
| 3 | `/result/[id]` | 结果页 | 风险灯 + 一句话 + 3 条解释 + 语音 + 「告诉家人」按钮 |
| 4 | `/card/[id]` | 家人卡预览 | Canvas 大图 + 长按保存提示 + 二维码回链 |

## 6. 3 个 Demo 案例

| ID | 风险灯 | 标题 | 一句话 |
|----|--------|------|--------|
| `demo-black` | 🔴 红 | 杀猪盘 | 这是骗子，钱进去就回不来 |
| `demo-gray` | 🟡 黄 | 虚假投资 | 高收益 = 高风险，先问子女 |
| `demo-white` | 🟢 绿 | 正常优惠 | 真的促销，可以买 |

## 7. 适老化规范 (设计 Token)

- **字号**：正文 ≥ 24px，标题 ≥ 48px，按钮文字 ≥ 32px
- **触控**：按钮 ≥ 88×88px，间距 ≥ 24px
- **对比度**：WCAG AAA，正文 ≥ 7:1
- **色板**：红 `#D7263D` / 黄 `#F4A300` / 绿 `#2E7D32` / 底 `#FAFAFA` / 字 `#1A1A1A`
- **字体**：系统中文字体，`font-weight: 600` 起步
- **动效**：全部 ≤ 300ms，尊重 `prefers-reduced-motion`
- **声音**：每页进入自动 TTS 朗读关键文案，可点「再听一遍」

## 8. 语音

- **输入**：Web Speech API (`webkitSpeechRecognition`) 可选，默认按钮触发
- **输出**：`speechSynthesis` 中文语音，语速 0.9，自动播放 + 重听按钮

## 9. 家人卡 (Canvas)

- 尺寸：1080×1920
- 顶部：风险灯大图 + 「家人你好」
- 中部：原图缩略 + 一句话 + 3 条解释
- 底部：二维码（指回 `/result/[id]`）+ 「看懂一下 · 给爸妈的反诈助手」
- 导出：`canvas.toBlob` → 下载 PNG

## 10. 双模型 + 离线兜底

```
用户提交
  ↓
有网？
  ├─ 是 → 豆包 API → 失败 → GPT-4o → 失败 → 离线 JSON
  └─ 否 → 离线 JSON (随机 3 个 demo 之一)
```

本次 Demo 默认走「离线 JSON」路径，确保现场零事故。

## 11. System Prompt (内容 Agent 产出)

要求：60+ 用户能秒懂，不用专业词，每条 ≤ 20 字。

## 12. 离线兜底 JSON

路径：`/public/demo-data.json`

```json
{
  "demo-black": { "level": "red", "title": "杀猪盘", "summary": "...", "points": ["...","...","..."] },
  "demo-gray":  { "level": "yellow", "title": "虚假投资", "summary": "...", "points": [...] },
  "demo-white": { "level": "green", "title": "正常优惠", "summary": "...", "points": [...] }
}
```

## 13. 故障应急

- 网断：自动 fallback 到离线 JSON
- 语音失败：显示「点这里再听一遍」按钮
- 图片传不上：用 `?demo=black` 等 URL 参数手动触发
- 整页崩：根 `error.tsx` + 大按钮「重来」

## 14. 90 秒演讲脚本 (内容 Agent 产出)

- 0-15s：开场痛点（爸妈被骗的真实数据）
- 15-30s：演示拍照 → 红灯结果
- 30-45s：演示分享家人卡
- 45-60s：断网演示（拔线）
- 60-90s：商业模式 + 团队 + Q&A 引子
